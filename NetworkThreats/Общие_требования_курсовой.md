**РТУ МИРЭА**

## Общие требования к проекту

- Проект должен решать практическую задачу (например, веб-приложение для управления данными: библиотека, магазин или CRM), демонстрируя навыки .NET 8+, ASP.NET Core и связанных технологий.
- Обязательное использование **Entity Framework Core** с подходом **CodeFirst**: модели сущностей в коде, миграции для создания/обновления БД (SQLite или SQL Server, PostgreSQL).
- Веб-интерфейс реализован на **Blazor Server** или **WebAssembly** с компонентами Razor, роутингом.
- Приложение полностью **контейнеризировано**: `Dockerfile` для сборки, `docker-compose.yml` для запуска (включая контейнеры для app, БД и опционально Redis).
- Исходный код размещён в **Git-репозитории** (GitHub/GitLab), желательно с ветками (`main`, `dev`, `feature/*`). `README.md` с инструкциями по запуску.

---

## Технические требования

### Архитектура
- Dependency Injection через `IServiceCollection`

### База данных
- EF Core модели с отношениями (1:1, 1:N, N:N)
- Репозитории
- `DbContext` с конфигурацией Fluent API
- Миграции применены и задокументированы

### Blazor
- Аутентификация (Identity) — опционально
- Страницы/компоненты с формами, валидацией (FluentValidation), модальными окнами
- Интеграция с EF через сервисы

### Docker
- Multi-stage build в `Dockerfile`
- Volumes для БД
- Healthchecks — опционально

### Тестирование (опционально)
- Unit-тесты (xUnit/NUnit) для бизнес-логики (coverage > 70%)
